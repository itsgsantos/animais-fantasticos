## 🚀 Technical Features

* **Object-Oriented Programming:** Built using JavaScript **Classes** to manage components like accordions, tabs, and modals in a clean, scalable way.
* **ES6 Modules:** Full implementation of `import` and `export` to maintain a decoupled and organized codebase.
* **Asynchronous Operations:** Utilization of **Async/Await** and **Fetch API** to integrate external data (Bitcoin price) and JSON files.
* **Custom Interaction Logic:** Advanced use of event delegation, scroll animations, and dynamic tooltips.
* **Data Validation:** Implementation of **Regular Expressions (RegEx)** for precise data formatting and form validation.
* **Mobile-First Responsiveness:** A fluid layout built with CSS Grid and Flexbox, optimized for all screen sizes.

## 🛠️ Tech Stack

- **Logic:** Vanilla JavaScript (ES6+, Classes, Modules)
- **Styling:** CSS3 (Variables, Animations, Grid/Flexbox)
- **Data:** Fetch API & REST Patterns
- **Versioning:** Git & GitHub

## 🧠 Key Learnings

This project served as a deep dive into **Vanilla JavaScript architecture**. 

**Core achievements:**
- Refactoring procedural code into **maintainable Classes**.
- Understanding the lifecycle of asynchronous requests and handling errors gracefully.
- Optimizing DOM performance by using efficient event listeners and delegation patterns.
- Mastering the use of **CSS Custom Properties** for consistent theme management.

## 💻 Getting Started

### Prerequisites
A modern web browser and a local server environment (like VS Code Live Server).

### Local Execution
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/itsgsantos/animais-fantasticos.git](https://github.com/itsgsantos/animais-fantasticos.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd animais-fantasticos
    ```
3.  **Run the project:**
    Open the project using a local server (required for ES6 Modules to function properly).
